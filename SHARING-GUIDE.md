### Part 3: 实战演示（20 分钟）

**演示 1：5 分钟快速体验**

```bash
# 1. 安装（1 分钟）
pip install qoder-cli

# 2. 初始化（1 分钟）
qoder init --lang java

# 3. 编写规范（2 分钟）
# 编辑 .openspec/features/hello.yaml

# 4. 生成代码（1 分钟）
qoder generate --all

# 5. 运行测试
mvn test
```

**演示 2：完整功能（10 分钟）**

```bash
# 1. 查看规范（2 分钟）
cat .openspec/features/user-register.yaml

# 2. 生成代码（2 分钟）
qoder generate --feature user-register

# 3. 查看生成的代码（3 分钟）
cat src/main/java/.../UserService.java
cat src/main/java/.../UserServiceImpl.java

# 4. 运行测试（3 分钟）
mvn test
```

**演示 3：前后对比（5 分钟）**

```
传统方式：
- 需求文档 5 页
- 开发 2 天
- 返工 3 次
- 总耗时：4 天

SDD 方式：
- 规范 1 个 YAML
- AI 生成 30 分钟
- 人工优化 1 小时
- 总耗时：2 小时

效率提升：10 倍！
```
