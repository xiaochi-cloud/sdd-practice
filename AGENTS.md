# AI 助手指南

> 给 AI 助手的开发指南

## 🎯 项目目标

构建规范驱动开发（SDD）实战平台，让 AI 自主编程成为日常。

## 📁 项目结构

```
sdd-practice/
├── README.md          # 主文档
├── AGENTS.md          # AI 助手指南（本文件）
├── LICENSE            # MIT 许可证
├── examples/          # 完整示例项目
├── templates/         # 规范模板
├── docs/              # 文档
└── advanced/          # 进阶内容
```

## 🤖 开发原则

1. **规范先行** - 先写规范，再写代码
2. **自动迭代** - 持续自动迭代优化
3. **质量第一** - 每个文件都要可运行
4. **文档完整** - 每个功能都有文档

## ⏰ 自动迭代

- 每 2 分钟自动迭代一次
- 每 30 分钟自动汇报进度
- 每天 08:50 晨间报告
- 每天 22:00 晚间总结
